## Project Overview
PhishSentinel is a dual-mode AI-powered phishing and fraud detection system.
It detects phishing URLs and suspicious email content using an ensemble ML model
(Random Forest + XGBoost) with SHAP explainability.

## Tech Stack
- Python, scikit-learn, XGBoost, SHAP, Flask, Pandas, NLTK
